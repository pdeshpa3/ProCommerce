# Procommerce - An Amazon Clone



## Tools and Technologies

Project is created with:
* JavaScript
* HTML
* CSS
* Node
* Express
* MongoDB

## Features

* View Products
* Add to Cart
* Checkout
* Payment via Paypal

## Admin features

* View Dashboard
* Create/Edit/Delete product
* View/Delete orders
* Analyze sales, users, products info
